# Hubot 9gag

A Hubot script that returns a random 9gag meme image (original author: EnriqueVidal).

## Installation

Run `npm install --save hubot-9gag`

Add **hubot-9gag** to your `external-scripts.json`:

```json
["hubot-9gag"]
```

## Usage

All these commands show a random 9gag image:
- hubot 9gag
- hubot 9gag me
- hubot 9gag jdoe
